# 广东财经大学-人机交互-实验三作业
该项目为人机交互课程的实验三作业，要求开发一个静态的app软件

## 注意事项

### 运行方式 
加载到Android Studio中运行即可。
### 程序环境
该项目在Android X 环境下开发，基于java语法开发
